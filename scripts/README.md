# Scripts Directory

This directory contains standalone scripts for various tasks related to the project.

## Directory Structure

```
scripts/
├── README.md          # This file
└── legacy/           # Legacy scripts preserved for reference
    ├── analisa_s4p_folgas.py      # S4P analysis script
    ├── malha_local.py             # Local mesh script
    └── pipeline_metaholo_auto.m   # MATLAB automation pipeline
```

## Legacy Scripts

The `legacy/` subdirectory contains older scripts that were used in earlier phases of the project. These are preserved for reference and historical purposes but may not be actively maintained.

### analisa_s4p_folgas.py
Analysis script for S-parameter data with tolerances/gaps.

### malha_local.py
Local mesh generation script.

### pipeline_metaholo_auto.m
MATLAB automation pipeline for meta-holography.

## Usage

Legacy scripts may require specific dependencies or configurations that are not documented here. Please refer to the git history or project documentation for more information on how these scripts were originally used.

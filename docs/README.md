## üí° Documentation Principles

All documentation in this project follows these principles:

1. **Bilingual**: Key documents include English and Portuguese (where applicable)
2. **Practical**: Focus on actionable information and examples
3. **Consistent**: Similar format across all modules
4. **Self-contained**: Each module documents its own functionality
5. **Up-to-date**: Documentation updates alongside code changes
